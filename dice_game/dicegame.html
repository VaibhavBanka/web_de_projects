Dice Game

Rules of the game

  1. Everytime you refresh the page the dice number changes for both player 1 and player 2.
  2. If the number of dots on player 1's dice is more player 1 wins.
  3. If the number of dots on player 2's dice is more player 2 wins.
  4. Else the match is drawn.
